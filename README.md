### App developed with [Typescript](https://www.typescriptlang.org/) and [Bootstrap](https://getbootstrap.com/).
The Joke App finds out if showing some jokes before starting the work day has an impact on a business employees. It will show a joke and let the user give a score to that joke. Every punctuation will be saved to an array of objects.

The main objective of this hypothetical project is to implement API calls and recieve it's data correctly, using TypeScript and Bootstrap CSS.

## Main Skills
### TypeScript
* Typing variables correctly.
* API fetching and promises.
### SCSS
* Using SCSS to customize elements and variables given by Bootstrap CSS library.
* Recreate the design idea given by the client.
* Responsive Design
